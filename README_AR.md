# مولد امتحانات الفيزياء — وصف ومميزات للمستهلك

هذا المشروع هو أداة ويب تفاعلية لتوليد أوراق امتحانية وملاحظات مراجعة في مادة الفيزياء بسرعة وبجودة عالية. تم تصميمه ليساعد المعلمين، والمدرسين الخصوصيين، وأولياء الأمور على إنشاء اختبارات مخصصة قابلة للطباعة أو التصدير بصيغ متعددة.

## الفكرة العامة

تقدّم الواجهة واجهة بسيطة تتيح للمستخدم تحديد إعدادات الامتحان (عدد الأسئلة، مستويات الصعوبة، نوع الأسئلة: اختيار من متعدد/إجابة قصيرة/مسائل حسابية)، ثم يقوم المحرك بتوليد ورقة امتحان متسقة ومنسقة جاهزة للطباعة.

## المميزات الرئيسية

- **توليد سريع ومباشر:** إنشاء ورق امتحان جديد خلال ثوانٍ مع تحكّم كامل في العنوان، الوقت، وعدد الأسئلة.
- **تنوع أسئلة ذكي:** يدعم أنماطاً مختلفة من الأسئلة (اختياري، مقالي، مسائل) مع إمكانية ضبط مستوى الصعوبة.
- **تنسيق جاهز للطباعة:** مخرجات متوافقة مع الطباعة وبتنسيق واضح ومنسق لسهولة الاستخدام في الصف.
- **تصدير ومشاركة:** إمكانيات تصدير للـ PDF أو نسخ المحتوى أو حفظ القوالب لإعادة الاستخدام.
- **تحرير غني للنصوص والمعادلات:** محرر نص غني يدعم إدخال معادلات رياضية وعرضها بشكل مناسب داخل الأسئلة.
- **إعدادات التصميم:** تخصيص مظهر ورقة الامتحان (خطوط، ترويسة، علامة مائية) ليناسب هوية المدرسة أو المعلم.

## فوائد للمستخدم (المستهلك)

- **توفر الوقت:** حذف الحاجة لإنشاء أسئلة يدوياً أو تنسيقها، خاصة لعوامل الوقت المحدود.
- **قابلية التخصيص:** امكانية تكييف الامتحان لصف دراسي معين أو مستوى طالب واحد.
- **احترافية المخرجات:** مظهر نهائي احترافي يرفع من مصداقية المادة التعليمية.

## كيفية الاستخدام السريعة

1. افتح واجهة التطبيق.
2. اختر نوع الورقة وعدد الأسئلة ومستوى الصعوبة.
3. اضبط إعدادات التصميم (اختياري).
4. اضغط "توليد" ثم عاين النتيجة.
5. صدّر الورقة إلى PDF أو اطبعها مباشرة.

## تخصيص وتوسعة

المشروع مبني بطريقة تتيح سهولة التعديل لإضافة قوالب أسئلة جديدة، ربط قواعد بيانات للأسئلة، أو إضافة دعم لغات أخرى.

## تواصل ودعم

للملاحظات أو طلب المزايا، افتح ملف `metadata.json` أو اتصل بمطوّر المشروع عبر القنوات الموجودة في المستودع.
